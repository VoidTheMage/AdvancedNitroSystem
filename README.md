# AdvancedNitroSystem

This mod improves everything related to nitro in the game.

## Features
* You have control over when you want to activate or deactivate your nitro, you can also perform nitro purges
* You can install and use nitro on bikes
* The mod has a customizable HUD to show you how much nitro your vehicle has, it was made to fit the game's original HUD
* Highly customizable, including 2 different modes, one behaves like an arcade racing game while the other behaves closer to the vanilla system
* Fixes NPCs using nitro
* It improves the behavior of the "All cars have nitro" cheat and the taxi missions reward
* It fixes the command GET_CURRENT_CAR_MOD for vehicles without a nitro dummy
* Compatible with controllers (requires GInput)
* The mod is open source
